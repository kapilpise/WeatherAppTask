How to use this application

  • Home screen

    • Here you can add location by clicking green location floating action button which will take you to map screen.

    • On Map screen you will be prompted with the dialog asking for location permission.

    • On Map screen you can drag and drop the marker on the desired location.

    • After dropping marker click on the marker which will open an info window of that location.

    • To book mark the location just click marker info window, which will book mark the location which you can see on Home screen.

    • From Home screen you can see bookmarked locations in a list and can delete this location by clicking on cross icon x.

  • Help screen

    • This the Help screen which is having details how this app works

  • Settings screen

    • You can set temp unit preference from here

    • Can reset cities data (clear all data of cities)
